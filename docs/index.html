🔧 Key Features

🗂 Project Dashboard

Organize all freelance projects in one place. Filter by category, view progress, and manage timelines effortlessly.

🧩 Task Management

Add, edit, sort, and prioritize tasks with a clean table view. Quick status toggles and deadline tracking included.

📌 Kanban Board

Visualize your task workflow. Drag-and-drop status changes across To-Do, In Progress, and Completed columns.

⚠️ Smart Alerts

Automatic warnings for overdue and urgent tasks — stay on top of what matters most.

📁 Category System

Assign custom categories to projects and tasks for faster filtering and clearer navigation.

📝 Local Data Storage

Runs entirely on your machine. No cloud syncing, no accounts. Just fast, secure, offline-first productivity.
